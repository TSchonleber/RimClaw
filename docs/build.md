# Build (draft)

## Requirements
- .NET Framework 4.8 SDK
- RimWorld assemblies (Assembly-CSharp.dll, UnityEngine.dll)

## Steps
1. Open `Source/OpenClaw/OpenClaw.csproj` in Visual Studio or Rider.
2. Add references to RimWorld `Managed` DLLs.
3. Build Release.
4. Copy output `OpenClaw.dll` to `Mod/Assemblies/`.
